# ARIES

This is a tui browser for the [https://en.wikipedia.org/wiki/Gemini_(protocol)](gemini protocol).

[assets/screenshot.png](aries homepage).
[assets/screenshot1.png](gemini protocol homepage).

# Installation

> [!WARNING]
> This has been tested and used only on linux.

```bash
git clone --depth=1 https://github.com/chachacollins/aries
cd aries
cargo build --release
cp ./target/release/aries $INSTALFOLDER
```

# Help and Documentation

To view the help page just press ctrl + h while anywhere in the browser.
